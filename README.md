## 🎯 Proje Hedefi

FocusGotchi, insanların telefon bağımlılığını veya dikkat dağınıklığını eğlenceli ve teşvik edici bir yapıya dönüştürmesi için tasarlandı. Odaklandıkça XP kazanır, kazandığınız bu XP ve altınlarla sevimli evcil hayvanınıza bakarsınız. Odak sürenizi bozarsanız hayvanınız acı çeker!

---

## 🔥 Çarpıcı Özellikler

- ⏳ **Akıllı Odak Sayacı (Pomodoro):** Belirlediğin süre boyunca (örn. 3dk, 5dk) seni telefonundan uzak tutan akıllı süre ölçer. Odak bozulmazsa XP ve altın kazandırır; yarıda kesilirse hayvanına ceza verir!
- 🦅 **Dinamik Evrim Sistemi:** Düzenli bakım, oyun ve odak modundan kazandığın XP ile karakterin Yumurtadan 🥚 ➡️ Yavruya 🐣, Çocuktan 🐥 ➡️ Usta Yetişkine 🦅 evrim geçirir!
- 🛒 **Premium Market:** Karakterini mutlu ve sağlıklı tutmak için kategorize edilmiş market. Hamburgerlerden şifalı iksirlere, hatta devasa XP veren **Kral Tacı'na 👑** kadar onlarca premium eşya!
- 🎮 **Oyun Salonu (5 Farklı Oyun):**
  - **Uzay Yakalayıcı:** Düşen nesneleri hızlı reflekslerle yakalama
  - **Hafıza Kartları:** Klasik beyin cimnastiği
  - **Hızlı Matematik:** Zamana karşı zihin testi
  - **Taş-Kağıt-Makas:** Klasik bir yüzleşme
  - **Renk Avcısı:** Doğru renkleri seçip hızı tepeye çıkarma
- 🏆 **Sürükleyici Başarım Sistemi:** Oynanış istatistiklerinizi analiz eden ve size çok aşamalı ödüller veren detaylı görev sistemi.
- 🌓 **Global Karanlık Mod (Dark Mode):** Tüm uygulamayı göz yormayan, premium bir gece temasıyla oynama şansı.
- 🗑️ **Hard-Reset (İlerlemeyi Sıfırla):** Çok fazla yorulduysan tek tıkla en başa dönüp yepyeni bir evcil hayvanla başlama macerası!

## 🛠️ Mimari ve Teknolojiler

Son derece modüler ve asenkron state modellemesine sahiptir:

- **React Native & Expo (SDK 54):** Her detayı özelleştirilmiş, hızlı UI deneyimi.
- **Context API:** 10'dan fazla iç içe çalışan eşzamanlı istatistiğin merkezi olarak tek kanaldan yönetilmesi.
- **AsyncStorage:** Uygulamayı kapatsanız dahi hiçlikten doğan verilerin korunması. İlerlemeniz sadece size ait.
- **React Navigation:** Yumuşak, alttan sekme ve akıcı bar animasyonları.

## 🚀 Projeyi Bilgisayarında Derle

Bu harika projeyi bilgisayarına kurmak veya kendi eklemelerini yapmak istersen aşağıdaki adımları izle:

### 1. Kaynak Kodları İndir ve Dizinine Gir
```bash
git clone https://github.com/Ali8183/EvcilHayvan-m.git
cd EvcilHayvan-m
```

### 2. Gerekli Kütüphaneleri Yükle
```bash
npm install
```

### 3. Uygulamayı Expo ile Başlat
```bash
npx expo start --clear
```

Açılan terminal ekranındaki **QR kodu** cep telefonundaki *Expo Go (Android/iOS)* uygulamasıyla okut ve anında oynamaya başla!
